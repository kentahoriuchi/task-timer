# Task Timer

A macOS menu bar app for tracking time spent on tasks.

## Requirements

- macOS 13 (Ventura) or later
- Swift 5.9+

## Setup

```bash
# Build
swift build

# Run
swift run
```

Or open `Package.swift` in Xcode and press Run.

## Usage

- Click the menu bar icon to open the task list
- Click a task to start/stop the timer
- Use "Add Task" to create a new task
- Use "Today" to see a summary of today's tracked time

## Adding a Remote

```bash
git remote add origin https://github.com/<your-username>/task-timer.git
git push -u origin main
```
